# Kolinoa

Python-IDE для Android с красивым интерфейсом: выбор версии Python (с поиском
и анимацией "скачивания"), терминал, и блокнот с кнопками Run/Stop и выводом.

Написано на **Kivy** — чистый Python, кроссплатформенно (запускается и на
десктопе для отладки, и собирается в APK под Android).

## Важно, что уже работает, а что — заглушка

- Меню выбора версии Python, поиск, анимация загрузки — **работают** (это
  фронтенд; сама "загрузка" сейчас имитируется прогресс-баром, потому что
  реальная установка второго интерпретера Python внутри Android-песочницы
  требует отдельного бэкенда, как у Termux — bootstrap-архивы под нужную
  архитектуру устройства).
- Терминал — **работает по-настоящему**: открывает `/bin/sh` и исполняет
  команды, которые ты вводишь.
- New file / Open file / блокнот с Run ▶ / Stop ■ и выводом — **работают**:
  реально сохраняет `.py`-файл, запускает его через `python3` и стримит
  stdout в интерфейс, Stop реально убивает процесс.

## Запуск на компьютере (для проверки интерфейса)

```bash
pip install kivy
python3 main.py
```

## Сборка в APK

Собрать в APK можно только на своей машине (Linux/WSL) — для этого нужен
Android SDK/NDK, которых нет в моей песочнице:

```bash
pip install buildozer cython
sudo apt install -y git zip unzip openjdk-17-jdk python3-pip autoconf \
    libtool pkg-config zlib1g-dev libncurses5-dev libncursesw5-dev \
    libtinfo5 cmake libffi-dev libssl-dev

buildozer -v android debug
```

Первая сборка долгая (buildozer сам скачает Android SDK/NDK). Готовый APK
появится в `bin/kolinoa-0.1-arm64-v8a_armeabi-v7a-debug.apk` — устанавливай
на телефон как обычно.

## Куда развивать дальше

- Настоящая установка разных версий Python "на лету" — по образцу
  termux-packages (bootstrap tar.xz под архитектуру устройства + распаковка
  в приватную директорию приложения).
- Подсветка синтаксиса в блокноте (например, через `pygments`).
- Полноценный терминал с ANSI-цветами и поддержкой интерактивных программ
  (потребует pty вместо простого subprocess, как в самом Termux).
